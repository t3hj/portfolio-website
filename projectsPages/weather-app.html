Weather App

A weather forecasting app built with JavaScript.

Project Overview

This Weather App project is a web application that provides weather forecasts for different locations. The app is built using JavaScript and leverages APIs to fetch real-time weather data. The project demonstrates the use of modern web technologies and API integration.

Welcome Screen

The app starts with a welcome screen where users can enter the name of a city to get the weather forecast. The interface is designed to be simple and user-friendly.

Weather Forecast

Once the user enters a city name, the app fetches the weather data from an API and displays the current weather, temperature, humidity, and other relevant information. The app also provides a 5-day weather forecast.

Key Features

  • Real-time Weather Data: The app fetches real-time weather data from a weather API.
  • 5-day Forecast: Provides a 5-day weather forecast for the entered location.
  • User-friendly Interface: The app features a simple and intuitive user interface.
  • Responsive Design: The app is designed to be responsive and works well on different devices.

Technical Details

JavaScript

The core logic of the app is implemented in JavaScript. It handles user input, API requests, and data rendering.

  • API Integration: The app uses the Fetch API to make requests to the weather API and retrieve data.
  • Data Rendering: The app dynamically updates the DOM to display the fetched weather data.
  • Error Handling: The app includes error handling to manage cases where the API request fails or the entered city is not found.

How to Use

  1. Open the app and enter the name of a city in the input field.
  2. Click the "Get Weather" button to fetch the weather data.
  3. The app will display the current weather and a 5-day forecast for the entered city.
