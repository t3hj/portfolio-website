Password Generator

This project is a Python-based password generator with customizable options and a strength indicator, built using Tkinter. It allows users to generate secure passwords with various configurations and save them for future use.

Features

  • Customizable password length and character types (letters, digits, punctuation).
  • Minimum character count settings for each character type.
  • Password strength indicator to evaluate the generated password's security.
  • Options to copy the password to the clipboard and save it to a file.

Technical Details

The Password Generator uses Python's Tkinter library to create a graphical user interface. It includes functions for generating passwords, updating the strength indicator, and saving passwords to a file.

How to Use

  1. Run the application and enter the desired password length.
  2. Select the character types to include (letters, digits, punctuation).
  3. Set the minimum counts for each character type if needed.
  4. Click the "Generate Password" button to create a password.
  5. Use the "Copy to Clipboard" button to copy the password or the "Save Password" button to save it to a file.
