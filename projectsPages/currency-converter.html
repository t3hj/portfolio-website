Currency Converter

This project is a Python-based currency converter that fetches real-time exchange rates and plots historical data. It allows users to convert between different currencies and visualize exchange rate trends over time.

Features

  • Real-time exchange rate fetching using an API.
  • Conversion between multiple currencies.
  • Historical exchange rate plotting for trend analysis.
  • Cache mechanism to store and reuse exchange rates.

Technical Details

The Currency Converter uses the requests library to fetch exchange rates from an API and matplotlib for plotting historical data. It also includes a caching mechanism to reduce API calls and improve performance.

How to Use

  1. Run the application and enter the amount to convert.
  2. Select the currencies to convert from and to.
  3. Optionally, enter a date to fetch historical exchange rates.
  4. Click the "Convert" button to see the result.
  5. Use the plotting feature to visualize exchange rate trends over a specified date range.
